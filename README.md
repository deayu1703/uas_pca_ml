## 📌 Dataset

Dataset yang digunakan adalah **WineQT Dataset**, yang berisi berbagai karakteristik kualitas anggur.

Target pada dataset adalah:

- `quality`

Sedangkan fitur yang digunakan dalam proses PCA adalah:

- Fixed Acidity
- Volatile Acidity
- Citric Acid
- Residual Sugar
- Chlorides
- Free Sulfur Dioxide
- Total Sulfur Dioxide
- Density
- pH
- Sulphates
- Alcohol
